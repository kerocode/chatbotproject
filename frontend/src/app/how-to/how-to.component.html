How it works

tensorflow 2.0 made developing such powerful deep learning model, a very easy task. having build-in Keras api inside tensorflow, make develop, train, run prediction using model in less then 100 line of code

  • first, design model, in our case we try to build Convolutional neural network (CNN), so i added multiple conv2D layers followed by Max Booling layer
  • second, we need to compile model and define optimizer and loss function.
  • finally, we need to train model.
  • to run prediction right from browser, we need to convert model from keras (.h5) to tensorflowjs (.json) using converter utilities method
  • to use model in browser, we need to include tensorflowjs script tag and use loadLayersModel to load model
  • now we have a complete running model that you can use to predict
